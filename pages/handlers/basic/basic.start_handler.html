Start Handler

Module defines the start handler for the Telegram bot using the aiogram framework.

async handlers.basic.start_handler.cmd_start(message: Message, state: FSMContext, i18n: I18nContext) → None

Handle the /start command.

This function is an asynchronous handler for the /start command in a Telegram bot. It clears the current state, sends a start message to the user with a menu keyboard, and sets the state to ‘start_menu’.

Parameters:

  • message (types.Message) – The incoming message object from the user.

  • state (FSMContext) – The finite state machine context for managing user states.

  • i18n (I18nContext) – The internationalization context for managing translations.
