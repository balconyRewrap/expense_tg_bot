Database Config

Module configures the database connection for the tg_bot_funko project.

It uses environment variables to set up the PostgreSQL database connection parameters and creates an asynchronous SQLAlchemy engine and session maker.

database.config.PGSQL_HOST

The hostname of the PostgreSQL server.

database.config.PGSQL_PORT

The port number of the PostgreSQL server.

database.config.PGSQL_DB

The name of the PostgreSQL database.

database.config.PGSQL_USER

The username for the PostgreSQL database.

database.config.PGSQL_PASSWORD

The password for the PostgreSQL database.

database.config.DATABASE_URL

The constructed database URL for the PostgreSQL connection.

Type:

str

database.config.engine

The asynchronous SQLAlchemy engine.

Type:

sqlalchemy.ext.asyncio.AsyncEngine

database.config.async_session_maker

The asynchronous session maker.

Type:

sqlalchemy.ext.asyncio.async_sessionmaker
